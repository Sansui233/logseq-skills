## Logseq Skills

Some logseq skills for maintenaince. Mostly based on file system. And Chinese first.

- `logseq-file-journal-entry` - Record scattered learning notes (journal-first workflow)
- `logseq-file-page-maintenance` - Maintain specific wiki entries (skip journal)
- `logseq-file-format-validator` - Validate Logseq markdown format
- `logseq-file-asset-manager` - Manage images, PDFs, documents, drawings
- [logseq-plugin-api](https://github.com/openclaw/skills/tree/main/skills/juanirm/logseq) Access local Logseq instance via http api.
